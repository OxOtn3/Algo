# Algo
基于 `Python3` 整理的一些算法板子，包括：
## 图论
- `Floyd`
- `Bellman-Ford`
  - 以及 `SPFA` 优化
- `Dijkstra`
  - 朴素
  - 堆优化
- `Topological Sort`
  - BFS
  - DFS
- `Kruskal`
- `Prim`

## 前缀树
- 小写英文字母的 `Trie`

## 并查集
- `Union-Find` (find优化)

## 树状数组
- `Fenwick-Tree`

## 排序
- `Heap Sort`
- `Merge Sort`
- `Quick Sort`
  - 以及 `Quick Select`
- `Bucket Sort`
- `Radix Sort`
- `Shell's Sort`

## 二叉树
- 迭代遍历（基于栈）
  - 前序
  - 中序
  - 后序
- `Morris`

## 蓄水池抽样
- `Reservoir Sampling`
  - 分布式场景下
  - https://www.jianshu.com/p/7a9ea6ece2af

## 数学
- `Fast Power` 快速幂
- `Newton's Method` 牛顿迭代法求平方根
- `Sieve of Eratosthenes` 埃氏筛法求素数

## 复合数据结构
- `LFU`
- `Max Frequency Stack` 最大频率栈
  - `pop()` 操作从堆栈中弹出出现频率最高的元素
- `Randomized Set` 
  - O(1) 时间插入、删除、获取随机元素

## 计算器
- `Calculator`
  - 加减乘除
  - 括号
